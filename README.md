# gitlab-user-projects-button

Tampermonkey script that add link to user projects in top bar

## Usage

1. Install the Tampermonkey browser extension (see links below)
2. Access the Tampermonkey settings and proceed with the installation [script](https://github.com/PixxxeL/gitlab-user-projects-button/raw/refs/heads/master/gitlab-user-projects-button.user.js)

## Полезные ссылки

* [Tampermonkey for Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
* [Tampermonkey for Chrome-like](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) (Chrome, YandexBrowser, Vivaldi, Chromium, etc)
